Question

To evaluate the trigonometric functions by using the unit circle, it can be helpful to know the following:

function value
sine \(y\)
cosine \(x\)
tangent \(\frac{y}{x}\)
cosecant \(\frac{1}{y}\)
secant \(\frac{1}{x}\)
cotangent \(\frac{x}{y}\)

The unit circle below has a radius of length 1. The coordinates of special angles are shown.


Determine \(\sec\left(\theta\right)\). You might need to rationalize a denominator and/or simplify a fraction.

Solution

We know that \(\sec\left(\theta\right)=\frac{1}{x}\). In this case, \(\theta=\frac{11\pi}{6}\) and \(x=\frac{\sqrt{3}}{2}\) and \(y=\frac{-1}{2}\). So

\[\sec\left(\frac{11\pi}{6}\right) = \frac{1}{\frac{\sqrt{3}}{2}}\]

Simplify the fraction. You might need to rationalize a denominator.

\[\sec\left(\frac{11\pi}{6}\right) = \frac{2\sqrt{3}}{3}\]

It should be mentioned that the fact that \(\theta=\frac{11\pi}{6}\) is not necessary to answer this question.
