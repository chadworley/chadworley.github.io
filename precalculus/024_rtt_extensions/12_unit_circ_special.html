Question

To evaluate the trigonometric functions by using the unit circle, it can be helpful to know the following:

function value
sine \(y\)
cosine \(x\)
tangent \(\frac{y}{x}\)
cosecant \(\frac{1}{y}\)
secant \(\frac{1}{x}\)
cotangent \(\frac{x}{y}\)

The unit circle below has a radius of length 1. The coordinates of special angles are shown.


Determine \(\cot\left(\theta\right)\). You might need to rationalize a denominator and/or simplify a fraction.

Solution

We know that \(\cot\left(\theta\right)=\frac{x}{y}\). In this case, \(\theta=\frac{7\pi}{4}\) and \(x=\frac{\sqrt{2}}{2}\) and \(y=\frac{-\sqrt{2}}{2}\). So

\[\cot\left(\frac{7\pi}{4}\right) = \frac{\frac{\sqrt{2}}{2}}{\frac{-\sqrt{2}}{2}}\]

Simplify the fraction. You might need to rationalize a denominator.

\[\cot\left(\frac{7\pi}{4}\right) = -1\]

It should be mentioned that the fact that \(\theta=\frac{7\pi}{4}\) is not necessary to answer this question.
