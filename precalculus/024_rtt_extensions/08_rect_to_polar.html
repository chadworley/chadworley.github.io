Question

A point on a Cartesian plane is given in rectangular coordinates: \[x = -1.57\] \[y=-3.45\]


Please determine the polar coordinates (\(r\) and \(\theta\)), enforcing \(0\le\theta<2\pi\).

  • The radius: \(r=\) ##ANSWER1##
  • The angle: \(\theta=\) ##ANSWER2##

Solution

First, get the radius, using Pythagorean Theorem.

\[r = \sqrt{(1.57)^2+(3.45)^2}\]

\[r \approx 3.79\]

Finding the reference angle by using the arctan function with the quotient of the absolute values of the rectangular coordinates.

\[\phi ~=~ \arctan\left(\frac{3.45}{1.57}\right)\]

\[\phi ~=~ 1.1437322\]


Because the point is in quadrant III, we add \(\pi\) and \(\phi\) to get \(\theta\). \[\theta = \pi+\phi\]

\[\theta\approx4.29\]

Unfortunately, you cannot simply use \(\arctan\left(\frac{-3.45}{-1.57}\right)\) to get the angle \(\theta\). This is because tangent is not a one-to-one function, so arctangent has a restricted range, from \(\frac{-\pi}{2}\) to \(\frac{\pi}{2}\).
