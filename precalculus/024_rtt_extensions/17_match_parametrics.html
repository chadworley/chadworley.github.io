Question

Match the parametric equations to their graphs.


  1. The following parametric equations match graph ##ANSWER1## \[x=5\cos(t)\] \[y=4\sin(t)\] \[0\le t \le 2\pi\]
  2. The following parametric equations match graph ##ANSWER2## \[x=(6+\sin(7t))\cos(t)\] \[y=(6+\sin(7t))\sin(t)\] \[0\le t \le 2\pi\]
  3. The following parametric equations match graph ##ANSWER3## \[x=-5+2\cos(t)\] \[y=-4+2\sin(t)\] \[0\le t \le 4\pi\]
  4. The following parametric equations match graph ##ANSWER4## \[x=5\sin(8t)\cdot\sin(5t)\] \[y=5\sin(2t)\cdot\sin(6t)\] \[-4\pi\le t \le 4\pi\]
  5. The following parametric equations match graph ##ANSWER5## \[x=6\cos(t)+4\cos(9t)\] \[y=6\sin(t)+4\sin(9t)\] \[0\le t \le 2\pi\]
  6. The following parametric equations match graph ##ANSWER6## \[x=\cos(t)\] \[y=t\] \[0\le t \le 2\pi\]
  7. The following parametric equations match graph ##ANSWER7## \[x=e^t\cos(8t)\] \[y=e^t\sin(8t)\] \[0\le t \le 2\pi\]
  8. The following parametric equations match graph ##ANSWER8## \[x=t\cos(2t)\] \[y=t\sin(2t)\] \[-2\pi\le t \le 2\pi\]

Solution

Graph using using Desmos or a similar graphing utility.

In Desmos, to graph parametric equations, use the same notation you’d use for a point, but instead of \((x,y)\), put two expressions with \(t\) as the independent variable.

For example, the first one, you’d type:

\[{\large(}5\cos(t),~~4\sin(t){\large)}\] And then set the proper domain of \(t\).
