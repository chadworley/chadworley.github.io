Question

A function \(f\) is graphed below.


  • If \(f(x)=4\), then find \(x\). In other words, evaluate \(f^{-1}(4)\).

Solution

Find the correct point on the graph where \(f(x)=4\).


\[f(3)=4\]

Thus,

\[f^{-1}(4) ~=~ 3\]
