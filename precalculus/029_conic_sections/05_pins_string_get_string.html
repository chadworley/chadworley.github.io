Question

A gardener is making an ellipse using the pins and string method. She wants the length of the major axis to be 16.4 meters and the length of the minor axis to be 16 meters.

How long should the string be?

Solution

Surprisingly, the length of the major axis exactly matches the length of the string. Both of them are a length \(2a=16.4\).
