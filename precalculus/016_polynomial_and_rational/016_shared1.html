  1. Question

    Let polynomial \(p(x)=2 x^{4} - 7 x^{2} + x - 3\).

    Use synthetic division to divide \(p(x)\) by \(x + 3\).



    Solution

    Begin by filling the top row with the coefficients of the polynomial. Remember to use 0 whenever a term is missing.

    Then, since the denominator is \(x + 3\), fill the leftmost element with \(-3\).

    2 0 -7 1 -3
    -3

    Drop the leading coefficient to the bottom row.

    2 0 -7 1 -3
    -3
    2

    Multiply the -3 by the 2 to get -6.

    2 0 -7 1 -3
    -3 -6
    2

    Add 0 and -6 to get -6.

    2 0 -7 1 -3
    -3 -6
    2 -6

    Continue to alternate between multiplication (by -3) and addition.

    2 0 -7 1 -3
    -3 -6 18 -33 96
    2 -6 11 -32 93

    The last number is called the remainder. One way to express the answer is with an identity (an equation that’s true for all \(x\)):

    \[\frac{2 x^{4} - 7 x^{2} + x - 3}{x + 3} ~=~ 2 x^{3} - 6 x^{2} + 11 x - 32 + \frac{93}{x + 3}\]

    From the remainder theorem, we also know that \(p(-3) = 93\). This can be seen by multiplying both sides by x + 3 and pluggling in \(x=-3\).

    \[p(x) ~=~ 2 x^{4} - 7 x^{2} + x - 3 ~=~ (2 x^{3} - 6 x^{2} + 11 x - 32)(x + 3) + (93) \]

    \[p(-3) ~=~ 2 (-3)^{4} - 7 (-3)^{2} + (-3) - 3 ~=~ (\ldots)(0) + (93) \]

    \[p(-3) ~=~ 93\]
