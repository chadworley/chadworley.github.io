Question

Which graph shows a correct pair of \(f(x)\) as a blue curve and \(f'(x)\) as a red curve? Remember the output (\(y\)) of the red derivative function should match the instantaneous slope of the blue curve for all values of \(x\).


Answerlist

  • A
  • B
  • C
  • D

Solution

The correct answer is A. Notice that when blue has positive slope, red has positive \(y\). When blue has negative slop, red has negative \(y\).

Probably the easiest characteristic to check first is where blue has a peak or valley (so slope equals 0), the red curve should have an \(x\) intercept.
