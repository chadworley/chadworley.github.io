  1. Question

    A geometric sequence has a common ratio between subsequent terms (remember an arithmetic progression has a common difference between terms).

    Let \(g_{n}\) represent the \(n\)th value in a geometric progression with the following recursive definition: \[g_1 = 7\] \[g_{[n+1]} = g_{[n]}\cdot 2\] where \(n\) is any natural number. Using this recursive definition can produce the next few values. \[g_2 ~=~ g_1 \cdot 2 ~=~ 7\cdot2 ~=~ 14\] \[g_3 ~=~ g_2 \cdot 2 ~=~ 14\cdot2 ~=~ 28\] \[g_4 ~=~ g_3 \cdot 2 ~=~ 28\cdot2 ~=~ 56\]

    In general, if a geometric progression has an initial term \(g_1\) and a common ratio \(r\), we can write an explicit formula to find the \(n\)th value.

    \[g_n = g_1r^{n-1} \]

    • Evaluate \(g_{9}\)

    Solution

    Use the explicit formula.

    \[g_n = g_1r^{n-1} \]

    \[g_{9} = (7)(2)^{9-1} \]

    \[g_{9} = (7)(2)^{8} \]

    \[g_{9} = (7)(256) \]

    \[g_{9} = 1792 \]
