  1. Question

    A landscape architect has 2 spots to plant 2 different trees. The landscape architect has 22 different trees available. How many configurations are possible?


    Solution

    This scenario describes a permutation problem.

    \[{_nP_r} = \frac{n!}{(n-r)!}\] \[n=22\] \[r = 2 \]

    So, the answer is 462.



  2. Question

    From a set with 15 elements, how many subsets with 4 elements are possible? (A set is an orderless, nonrepeating collection of elements.)


    Solution

    This scenario describes a combination problem.

    \[{_nC_r} = \frac{n!}{(n-r)! \cdot r!}\] \[n=15\] \[r = 4 \]

    So, the answer is 1365.

  21. Question

    In some situation, each trial has 0.36 probability of success. There will be 7 trials. (Thus the number of successes will follow a binomial distribution.)

    What is the probability that the number of successes is less than \(3\).


    Solution

    This is a binomial distribution, so use the appropriate formulas. \[P[X=x] ~=~ _n\text{C}_x \cdot p^x(1-p)^{n-x}\] where \(p\) is the probability of success on each trial, \(x\) is a specific number of successes, \(n\) is the number of trials, and \(C\) is the combinations operator (so that \({_n}C_x = \frac{n!}{(n-x)! \, x!}\)). Some people prefer using \(q\) as the probability of failure, such that \(q=1-p\). \[P[X=x] ~=~ _n\text{C}_x \cdot p^xq^{n-x}\]

    You can generate the probability distribution.

    x P[X=x]
    0 0.0439805
    1 0.1731731
    2 0.2922296
    3 0.2739652
    4 0.1541054
    5 0.0520106
    6 0.0097520
    7 0.0007836

    You will also need to add mutually exclusive probabilities (when multiple \(x\)-values satisfy the probabilitys condition). It is also helpful to be aware of the complement rule.

    \[P(X<3) = 0.5093831\]
