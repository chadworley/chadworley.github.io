Question

To generate the plot below, four parameters (\(A\), \(B\), \(C\), and \(D\)) where randomly selected and plugged into the following sinusoidal function:

\[y(t) = A\sin(Bt+C)+D\]

where \[A>0\] and \[0\le C < 2 \pi\]


The first maximum is \((2.26, 0.39)\) and the first minimum is \((1.12, -0.69)\). Determine the values of the parameters (rounded to the nearest hundredth).

  • \(D =\) ##ANSWER4##

Solution

  • \(A = 0.54\)
  • \(B = 2.76\)
  • \(C = 1.63\)
  • \(D = -0.15\)

The easiest parameters to find are \(A\) and \(D\). With the given parameterization, \(A\) is the (semi-)amplitude and \(D\) is the vertical shift of the midline.

\[A ~=~ \frac{(0.39)-(-0.69)}{2} ~=~ 0.54\]

\[D ~=~ \frac{(0.39)+(-0.69)}{2} ~=~ -0.15\]


In the given parameterization, \(B\) represents the angular frequency. It is inversely proportional to the period, which is easy to determine from the 2 given points. The two marked points are half a period away from each other (in time). \[P ~=~ 2\cdot(2.26-1.12) ~=~ 2.28\]


You can calculate the angular frequency by dividing \(2\pi\) by the period.

\[B = \frac{2\pi}{2.28} = 2.755783\] We round to the hundredths place.

\[B \approx 2.76\]

With the given parameterization, \(C\) represents the phase (in radians), representing how far into the cycle the wave is when \(t=0\). It will be proportional to the horizontal shift. The horizontal shift is to the left, because a positive \(C\) is added inside the argument. The parent function (sine) has a point at the origin with a positive slope. Find the analogous point (on the midline with positive slope) to the left of the vertical axis.

First, determine the quarter period is \(P/4 = 0.57\). Then, recognize that the key point is 3 quarter periods before the first given point. So, to find the \(t\) value of the key point: \[1.12-3\cdot0.57 ~=~ -0.59\] And so the key point is at \((-0.59, -0.15)\).


The sine function was shifted 0.59 time units to the left. If we divide 0.59 by the period and multiply by \(2\pi\), it tells us the phase.

\[C = \frac{0.59}{2.28}\cdot 2\pi = 1.625912\]

Round to the nearest hundredth:

\[C \approx 1.63\]
