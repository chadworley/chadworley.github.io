26_predict_with_trendline


  1. Question

    x y
    48.4 122.3
    38.8 101.8
    48.3 125.4
    48.9 133.5
    45.0 117.3
    44.6 119.7

    Find the trendline using a simple linear regression. Do not assume the \(y\)-intercept is 0.

    Using the linear model, predict \(y\) when \(x=59\).


    Solution

    • Copy the \(x\) and \(y\) values into a spreadsheet.
    • Highlight the data. Insert chart \(\to\) scatterplot.
    • Under Setup, click Use column A as labels
    • Under Customize \(\to\) Series, click Trendline
    • Change the Label to Use Equation

    The chart+trendline provide a nice visual, and an approximate linear model. To get more exact values for the slope and y-intercept, use the =linest() function. The first argument is the y values, and the second argument is the x values. For the third argument (calculate_b), use true. For the fourth argument (verbose), use false.

    A B C
    1 x y
    2 48.4 122.3 =linest(B2:B7, A2:A7, TRUE, FALSE)
    3 38.8 101.8
    4 48.3 125.4
    5 48.9 133.5
    6 45 117.3
    7 44.6 119.7


    That linest will provide two numbers: the slope 2.5987408 and the intercept 1.3241689.

    So, our linear model is: \[y = (2.5987408)x + (1.3241689) \]

    Plug in the given value of \(x=59\) for prediction.

    \[y ~=~ (2.5987408)(59) + (1.3241689) ~=~ 154.6498777\]
