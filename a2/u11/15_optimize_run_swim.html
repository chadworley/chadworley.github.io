Question

A lifeguard wants to save a struggling swimmer as soon as possible. The lifeguard can run along the beach at 2.9 m/s and swim at 1.8 m/s. The lifeguard will run \(x\) meters and swim \(\sqrt{(170-x)^2+40^2}\) meters.


Thus, the time is a function of \(x\).

\[t(x) ~=~ \frac{x}{2.9} + \frac{\sqrt{(170-x)^2+40^2}}{1.8}\]

Find the value of \(x\) (in meters) that minimizes the amount of time. The tolerance is \(\pm 0.1\) meters.

Solution

Use a graphing utility to find the minimum.


So the optimal value of \(x\) equals 138.33 in order to get to the struggling swimmer as quickly as possible.
