  1. Question

    A mass-spring system oscillates over time. The position (in meters) of the mass is plotted as a function of time (in seconds).


    Choose a trigonometric function (\(g\)), and the parameters (\(A\), \(P\), and \(D\)), to model this oscillatory motion.

    \[y~=~A\cdot g\left(\frac{2\pi}{P}t\right)+D\]

    • \(g=\)
    • \(A=\)
    • \(P=\)
    • \(D=\)


    Solution

    • The amplitude is 3 meters.
    • The period is 8 seconds.
    • The midline is at -2 meters.

    The amplitude is the absolute value of \(A\). \[|A|~=~3\]

    1. If when \(t=0\) (at \(y\)-axis) the wave is at midline and is increasing, then \(A>0\) and \(g=\sin\).
    2. If when \(t=0\) (at \(y\)-axis) the wave is at midline and is decreasing, then \(A<0\) and \(g=\sin\).
    3. If when \(t=0\) (at \(y\)-axis) the wave is at maximum, then \(A>0\) and \(g=\cos\).
    4. If when \(t=0\) (at \(y\)-axis) the wave is at minimum, then \(A<0\) and \(g=\cos\).

    \[A=-3\] \[g=\cos\]

    The vertical shift is determined by the midline at \(y=-2\).

    \[D=-2\]

    So,

    \[y ~=~ -3\cdot\cos\left(\frac{2\pi}{8}t\right)-2\]

    • \(g=\cos\)
    • \(A=-3\)
    • \(P=8\)
    • \(D=-2\)

    Extra stuff

    Of course, we would typically simplify the fraction.

    \[y ~=~ -3\cdot\cos\left(\frac{\pi}{4}t\right)-2\]

    The value \(\frac{\pi}{4}\approx0.7853982\) is called the angular frequency, which weâ€™ve been representing with \(B\).

    \[B=\frac{2\pi}{P}\] \[P=\frac{2\pi}{B}\]
