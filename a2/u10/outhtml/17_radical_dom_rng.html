  1. Question

    Consider the radical function \(f\) defined below.

    \[f(x)~=~-\sqrt{x-8}-2\]

    We wish to indicate the domain and range of function \(f\).

    • Domain:
      • As an inequality:
      • In interval notation:
    • Range:
      • As an inequality:
      • In interval notation:


    Solution

    A fully parameterized version of the given function could be written as:

    \[f(x)~=~a\sqrt{b(x-h)}+k\]

    In our case, we were given \(f(x)~=~-\sqrt{x-8}-2\), so

    \[a = -1\] \[b = 1\] \[h = 8\] \[k = -2\]

    Because \(a=-1\) and \(b=1\), the parent function is reflected vertically, so from the starting point at (8, -2), the curve goes down and to the right.


    The starting point is indeed included in the domain and range because,

    \[-2~=~-\sqrt{(8)-8}-2\]

    Thus the finite boundaries should be inclusive. This means we use the “or equal” symbols in the inequalities and “[” or ”]” in interval notation on the finite boundary. Remember, domain indicates the values of \(x\) over which the function exists, and range indicates the values of \(y\) over which the function exists.

    • Domain:
      • As an inequality: x ≥ 8
      • In interval notation: [8, ∞)
    • Range:
      • As an inequality: y ≤ -2
      • In interval notation: (-∞, -2]
