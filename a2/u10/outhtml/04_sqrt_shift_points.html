04_sqrt_shift_points


  1. Question

    The function \(f(x)~=~\sqrt{x}\) has some integer input-output pairs (see Diophantine equations). The first 7 are listed and graphed below.

    \(x\) \(y=f(x)\)
    0 0
    1 1
    4 2
    9 3
    16 4
    25 5
    36 6


    A shifted function \(g\) is defined as \(g(x)=\sqrt{x+1}-4\). Find the corresponding points of integer input and integer output on function \(g\).

    \(x\) \(y=g(x)\)


    Solution

    Function \(g\) can be thought of a shifted version of \(f\), shifted horizontally by -1 and vertically by -4.

    Graph of function \(g\)


    \(x\) \(y=f(x)\)
    -1 -4
    0 -3
    3 -2
    8 -1
    15 0
    24 1
    35 2
