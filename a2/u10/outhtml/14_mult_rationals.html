  1. Question

    Counterintuitively, when multiplying fractions, it is often helpful to start by factoring and to eliminate common factors. For example,

    \[\frac{77}{6}\cdot\frac{4}{35}~=~\frac{11\cdot7\cdot2\cdot2}{2\cdot3\cdot5\cdot7}~=~\frac{11\cdot2}{3\cdot 5} ~=~ \frac{22}{15}\]

    A similar technique is helpful for multiplying rational expressions.

    It is possible to enforce the following equation to be true for all values of \(x\) (and thus make an identity) by setting the four parameters: \(a\), \(b\), \(c\), and \(d\). Note: technically, there will be 2 specific values of \(x\) that cause the left side to be undefined but not the right side because we will be filling some removable discontinuities.

    \[\frac{x^{2}-{13}x+{36}}{x^{2}+{8}x+{7}}\cdot\frac{x^{2}+{5}x-{14}}{x^{2}-{14}x+{45}} ~\equiv~ \frac{x^2+ax+b}{x^2+cx+d}\]

    Find \(a\), \(b\), \(c\), and \(d\) to set up the identity above.

    • \(a=\)
    • \(b=\)
    • \(c=\)
    • \(d=\)


    Solution

    Start with the given product.

    \[\frac{x^{2}-{13}x+{36}}{x^{2}+{8}x+{7}}\cdot\frac{x^{2}+{5}x-{14}}{x^{2}-{14}x+{45}}\]

    Instead of starting by multiplying the numerators and multiplying the denominators, it can be helpful to start by factoring each expression.

    \[\frac{(x-{9})(x-{4})}{(x+{7})(x+{1})}\cdot\frac{(x-{2})(x+{7})}{(x-{9})(x-{5})}\]

    Write as a single fraction.

    \[\frac{(x-{9})(x-{4})(x-{2})(x+{7})}{(x+{7})(x+{1})(x-{9})(x-{5})}\]

    Eliminate common factors. (This step removes the removable discontinuities… fills the holes.)

    \[\frac{(x-{4})(x-{2})}{(x+{1})(x-{5})}\]

    Expand both the numerator and denominator using the FOIL method.

    \[\frac{x^{2}-{6}x+{8}}{x^{2}-{4}x-{5}}\]

    Identify the values of the parameters to enforce the identity.

    \[\frac{x^{2}-{6}x+{8}}{x^{2}-{4}x-{5}} ~\equiv~ \frac{x^2+ax+b}{x^2+cx+d}\]

    So,

    \[a=-6\] \[b=8\] \[c=-4\] \[d=-5\]
