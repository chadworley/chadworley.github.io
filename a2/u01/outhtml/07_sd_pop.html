  1. Question

    A lottery machine (population) will produce random numbers with replacement. It has 4 balls, with the following numbers:

    \(x_i\)
    73
    64
    64
    61


    Please calculate the population standard deviation. To do this, first calculate the population mean.

    \[\mu = \frac{\sum\limits_{i=1}^n x_i}{n}\]

    Calculate the population variance by averaging the squared deviations. \[\text{variance} = \sigma^2 = \frac{\sum\limits_{i=1}^n (x_i-\mu)^2}{n} \]

    Determine the population standard deviation by taking the square root of the population variance. \[\sigma = \sqrt{\sigma^2} \]


    Solution

    Find the mean. \[\mu = \frac{73+64+64+61}{4} = 65.5 \]

    Find the variance.

    \[\sigma^2 = \frac{(73-65.5)^2+(64-65.5)^2+(64-65.5)^2+(61-65.5)^2}{4} \]

    \[\sigma^2 = \frac{(7.5)^2+(-1.5)^2+(-1.5)^2+(-4.5)^2}{4}\]

    \[\sigma^2 = \frac{56.25+2.25+2.25+20.25}{4}\]

    \[\sigma^2 = 20.25\] Calculate the standard deviation. \[\sigma = \sqrt{20.25} = 4.5 \]

    You might find it helpful to organize your work in a table.

    \(x_i\) \((x_i-\mu)\) \((x_i-\mu)^2\)
    73 7.5 56.25
    64 -1.5 2.25
    64 -1.5 2.25
    61 -4.5 20.25
    \(\sum\limits_{i=1}^{n} x_i\) \(\sum\limits_{i=1}^{n} (x_i-\mu)^2\)
    262 81
    \(\mu\) \(\sigma^2\)
    65.5 20.25
    \(\sigma\)
    4.5
