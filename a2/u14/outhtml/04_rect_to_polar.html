04_rect_to_polar


  1. Question

    A point on a Cartesian plane is given in rectangular coordinates: \[x = -3.86\] \[y=-1.6\]


    Please determine the polar coordinates (\(r\) and \(\theta\)), enforcing \(0\le\theta<2\pi\).

    • The radius: \(r=\)
    • The angle: \(\theta=\)


    Solution

    First, get the radius, using Pythagorean Theorem.

    \[r = \sqrt{(3.86)^2+(1.6)^2}\]

    \[r \approx 4.18\]

    Find the reference angle by using the arctan function with the quotient of the absolute values of the rectangular coordinates.

    \[\phi ~=~ \arctan\left(\frac{1.6}{3.86}\right)\]

    \[\phi ~=~ 0.3929502\]


    Because the point is in quadrant III, we add \(\pi\) and \(\phi\) to get \(\theta\). \[\theta = \pi+\phi\]

    \[\theta\approx3.53\]

    Unfortunately, you cannot simply use \(\arctan\left(\frac{-1.6}{-3.86}\right)\) to get the angle \(\theta\). This is because tangent is not a one-to-one function.

    Now, Desmos (and many other calculators) allow you to feed \(y\) and \(x\) as two seperate arguments into the arctangent function: \(\arctan(-1.6,-3.86)\). This will give you a correct angle, between \(-\pi\) and \(\pi\). If you get a negative angle, you can just add \(2\pi\) to get a coterminal angle between \(0\) and \(2\pi\), as the prompt asks for.
