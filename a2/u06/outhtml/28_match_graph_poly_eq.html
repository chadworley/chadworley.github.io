    Graph 1


    Which function matches Graph 1?

    • \(A(x)~=~-(x-3)^{2}(x-7)^{2}\)

    • \(B(x)~=~-(x-3)^{2}(x+7)^{2}\)

    • \(C(x)~=~-(x+3)(x-7)^{2}\)

    • \(D(x)~=~(x+3)(x+7)^{2}\)

    Graph 2


    Which function matches Graph 2?

    • \(A(x)~=~(x-2)(x+5)\)

    • \(B(x)~=~(x-2)^{2}(x+5)\)

    • \(C(x)~=~-(x+2)(x-5)\)

    • \(D(x)~=~-(x+2)^{2}(x-5)^{2}\)

    Graph 3


    Which function matches Graph 3?

    • \(A(x)~=~-(x-5)(x+9)\)

    • \(B(x)~=~(x-5)^{2}(x+9)\)

    • \(C(x)~=~-(x+5)(x-9)\)

    • \(D(x)~=~(x+5)(x+9)\)




    Solution

    • Graph 1 is \(A(x)~=~-(x-3)^{2}(x-7)^{2}\)
    • Graph 2 is \(C(x)~=~-(x+2)(x-5)\)
    • Graph 3 is \(C(x)~=~-(x+5)(x-9)\)

    Remember, each factor is \((x-r)\) where \(r\) is the root. If the factor is squared, we expect a bounce. Lastly, make sure to pay attention to the sign in front, which will dictate the end behavior.
