52_exponential_asymptote


  1. Question

    Consider the exponential function:

    \[f(x)~=~5.21^{x-8}-2\]

    Describe the function’s asymptote. The function \(f\) has a at .



    Solution

    Graph the function as \(y=f(x)\). Notice that as \(x\to-\infty\), we see \(y\to -2\). Thus, there is a horizontal asymptote at \(y=-2\). Notice this means the range is \((-2,\infty)\), so the possible outputs include all real numbers greater than -2.


    You might think it looks like there is also a vertical asymptote, but you’d be wrong. The curve has a domain of \((-\infty,\infty)\).
