  1. Question

    Consider the exponential function:

    \[f(x)~=~\log_{3.17}\left(x+8\right)+4\]

    Describe the functionâ€™s asymptote. The function \(f\) has a at .



    Solution

    Graph the function as \(y=f(x)\). Notice that as \(y\to-\infty\), we see \(x\to -8\). Thus, there is a vertical asymptote at \(x=-8\). Notice this means the domain is \((-8,\infty)\), so the possible inputs include all real numbers greater than -8.


    You might think it looks like there is also a horizontal asymptote, but youâ€™d be wrong. The curve has a range of \((-\infty,\infty)\).
