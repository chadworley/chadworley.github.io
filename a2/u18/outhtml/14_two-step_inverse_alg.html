  1. Question

    Let function \(f\) be defined with the algebraic formula below: \[f(x) = \frac{x}{3}+7\]

    Find an expression for the inverse function \(f^{-1}\).


    1. \(f^{-1}(x)~=~3(x-7)\)
    2. \(f^{-1}(x)~=~3(x+7)\)
    3. \(f^{-1}(x)~=~7x-3\)
    4. \(f^{-1}(x)~=~\frac{x}{7}-3\)

    Solution

    The original function tells us to divide by 3 and then add 7.

    The inverse function does the inverse operations in the reverse order, so it must subtract 7 and then multiply by 3.

    \[f^{-1}(x)=3(x-7)\]
