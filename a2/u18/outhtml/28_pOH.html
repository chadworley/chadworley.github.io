  1. Question

    In acid-base chemistry, there is a set of 4 equations that are used for conversion between 4 important measurements: \(\mathrm{[H^+]}\), \(\mathrm{pH}\), \(\mathrm{[OH^-]}\), and \(\mathrm{pOH}\). The square brackets are used to indicate concentration (usually as moles per liter), and pH and pOH are defined by the equations below.


    • If an aqueous solution has known concentration of hydrogen ions, \(\mathrm{\left[H^+\right]}=3.31\cdot 10^{-6}\) moles per liter, what is the pOH? The tolerance is \(\pm 0.1\).

    Solution

    There are two ways to solve this problem (clockwise or counterclockwise on the given diagram). If we go clockwise, we first calculate \(\mathrm{[OH^-]}\) and then calculate \(\mathrm{pOH}\). If we go counterclockwise, we first calculate \(\mathrm{pH}\) and then calculate \(\mathrm{pOH}\).

    Let’s first go clockwise. Plug \(\mathrm{\left[H^+\right]}=3.31\cdot 10^{-6}\) into the equation \(\mathrm{[H+]\cdot [OH^-]=10^{-14}}\) and solve for \(\mathrm{[OH^-]}\).

    \[\left(3.31 \cdot 10^{-6}\right)\cdot \mathrm{[OH^-]}=10^{-14}\]

    \[\mathrm{[OH^-]}=\frac{10^{-14}}{3.31\cdot 10^{-6}}\]

    \[\mathrm{[OH^-]}=3.0211\cdot 10^{-9}\]

    Use the definition of pOH.

    \[\mathrm{pOH}~=~-\log_{10}\left(\left[\mathrm{OH}^-\right]\right)\]

    \[\mathrm{pOH}~=~-\log_{10}\left(3.0211\cdot 10^{-9}\right)\]

    \[\mathrm{pOH}~=~8.519828\]

    Let’s now go counterclockwise. First use the definition of pH with the given concentration of hydrogen ions.

    \[\mathrm{pH}~=~-\log_{10}\left(\left[\mathrm{H}^+\right]\right)\]

    \[\mathrm{pH}~=~-\log_{10}\left(3.31 \cdot 10^{-6}\right)\]

    \[\mathrm{pH}~=~5.480172\]

    Now use the equation \(\mathrm{pH+pOH}=14\) to find \(\mathrm{pOH}\).

    \[5.480172+\mathrm{pOH}=14\] \[\mathrm{pOH}=14-5.480172\] \[\mathrm{pOH}=8.519828\]

    Either way we get approximately \(\mathrm{pOH}=8.5\).
