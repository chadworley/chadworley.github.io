36_inv_func_from_graph


  1. Question

    A function \(f\) is graphed below as \(y=f(x)\).


    The inverse of the function is often denoted with a superscripted \(-1\), so the inverse of \(f\) is \(f^{-1}\). This is an awful notation that should one day be fixed. In this context, the superscript is NOT an exponentiation. As stated, in this context the superscript of \(-1\) implies the inverse function, which is the function that undoes another function.

    • Evaluate \(f^{-1}(7)\).

    Solution

    Look up inverse functions.

    Find the correct point on the graph where \(f(x)=7\).


    \[f(8)=7\]

    Thus,

    \[f^{-1}(7) ~=~ 8\]
