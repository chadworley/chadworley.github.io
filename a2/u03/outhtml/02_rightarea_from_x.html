  1. Question

    Random variable \(B\) is normally distributed with mean \(\mu = 480\) and standard deviation \(\sigma = 150\). \[B \sim \text{N}(480,\,150) \] Evaluate \(P(B > 390)\).


    Solution

    0.7257

    Use an online app

    Go to either:

    • chadworley.github.io/a2/tools/normal.html
      • Set \(\mu\) to 480
      • Set \(\sigma\) to 150
      • Choose Right interval (radio button)
      • Set \(b\) to 390.
    • www.desmos.com/calculator
      • Type NORMALDIST(480,150)
      • Click Find Cumulative Probability (CDF).
      • Enter 390 as Min.

    video (using app)

    Using z table

    video (using z-table)

    First, draw a sketch. We can label the \(B\) axis by adding integer multiples of 150 to 480. We know to shade toward the right because large values of \(B\) satisfy the condition \(B>390\).


    We are given a specific \(b\) value as a boundary. (Remember, for random variables we use uppercase letters, but for specific values we use lowercase.) \[b = 390 \] We calculate the \(z\) value of the boundary. \[\begin{aligned} z &= \frac{b-\mu}{\sigma} \\\\ &= \frac{390-480}{150} \\\\ &= -0.6 \end{aligned}\]

    We have rephrased our problem into a standard normal probability problem, because \[P(B>390) ~=~ P(Z>-0.6) \]

    So, we just need to evaluate \(P(Z>-0.6)\).

    To do this, you need to remember that right-area events are complementary to left-area events. \[P(Z>-0.6) ~=~ 1-P(Z<-0.6) \]

    You can use a \(z\)-table.

    \(z\) \(P(Z<z)\)
    -0.62 0.2676
    -0.61 0.2709
    -0.6 0.2743
    -0.59 0.2776
    -0.58 0.281

    Thus, we find our answer. \[\begin{aligned} P(B>390) &= 1-0.2743 \\ &= 0.7257 \end{aligned}\]
