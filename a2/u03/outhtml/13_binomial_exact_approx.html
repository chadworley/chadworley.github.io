  1. Question

    Tyler has a 77% chance to hit each freethrow (so \(p=0.77\)). Each freethrow is independent.

    Each day, Tyler attempts \(n=30\) freethrows. Each success gets 1 point. Each failure gets 0 points.

    We can predict the distribution of points per day . We will use 2 strategies:

    • An exact binomial distribution.
    • A normal approximation.


    • We can set up the exact binomial in Desmos using binomialdist(30,0.77)

    To set up the normal approximation, we need to find the mean and the standard deviation.

    \[\mu = np\] \[\sigma = \sqrt{np(1-p)}\]

    • Once you get \(\mu\) and \(\sigma\), you can set up the normal approximation in Desmos using normaldist(\(\mu\),\(\sigma\))

    Please find the following probabilities:

    1. Using the exact binomial, calculate \(P(X<22.5)\).
    2. Using the normal approximation, calculate \(P(X<22.5)\).
    3. Using the exact binomial, calculate \(P(X>26.5)\).
    4. Using the normal approximation, calculate \(P(X>26.5)\).
    5. Using the exact binomial, calculate \(P(18.5<X<22.5)\).
    6. Using the normal approximation, calculate \(P(18.5<X<22.5)\).

    (Notice I already gave half-integer boundaries for the continuity correction.)



    Solution

    1. Exact: \(P(X<22.5)=0.383\)
    2. Approx: \(P(X<22.5)\approx0.397\)
    3. Exact: \(P(X>26.5)=0.062\)
    4. Approx: \(P(X>26.5)\approx0.070\)
    5. Exact: \(P(18.5<X<22.5)=0.355\)
    6. Approx: \(P(18.5<X<22.5)\approx0.374\)

    Find \(\mu\) and \(\sigma\) for normal approximation

    \[\mu ~=~ pn ~=~ (0.77)(30) ~=~ 23.1\]

    \[\sigma ~=~ \sqrt{np(1-p)} ~=~ \sqrt{(30)(0.77)(1-0.77)} ~=~ 2.3049946\]
