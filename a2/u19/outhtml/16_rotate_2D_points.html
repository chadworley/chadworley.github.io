  1. Question

    Matrix of original points (preimage)

    A list of 2D-cartesian points is stored as the preimage matrix. Each row represents one point.

    \[P = \begin{bmatrix} -7 & 6\\ 3 & 1\\ -3 & -3\\ 6 & -1\\ 2 & -5\\ -5 & 3\\ -6 & -7\\ 3 & -3\\ -1 & -5\\ -1 & -2 \end{bmatrix}\]

    Those points can be expressed in SageMath.

    P = matrix([[-7,6],[3,1],[-3,-3],[6,-1],[2,-5],[-5,3],[-6,-7],[3,-3],[-1,-5],[-1,-2]])

    If you wanted to visualize those points, you could use the point function.

    point(P,xmin=-10,xmax=10,ymin=-10,ymax=10,size=50,aspect_ratio=1)

    Rotation matrix

    We want to rotate the points by \(\theta=3.07\) radians counterclockwise. We will do this by matrix multiplication with the 2D rotation matrix. (We are using a right-operand rotation matrix. The standard approach is to use a left-operand rotation matrix, but I think - for complicated reasons - that weâ€™ll be better off using right-operand transformations.)

    \[R = \begin{bmatrix}\cos(\theta) & \sin(\theta) \\ -\sin(\theta) & \cos(\theta) \end{bmatrix}\]

    Image

    We can apply the rotation to the points by using matrix multiplication. Remember, matrix multiplication is not commutative. Let \(F\) represent a matrix holding the final positions of the image (after the transformation).

    \[F = P\cdot R\]

    After you calculate \(F\), you can plot the points in \(F\) with the following code (in SageMath):

    point(F,xmin=-10,xmax=10,ymin=-10,ymax=10,size=50)

    Which is the graph of the points in \(T\)? (Please excuse minor stylistic differences due to my graphs being made in R, not SageMath.)



    1. A
    2. B
    3. C
    4. D

    Solution

    P = matrix([[-7,6],[3,1],[-3,-3],[6,-1],[2,-5],[-5,3],[-6,-7],[3,-3],[-1,-5],[-1,-2]])
theta = 3.07
R = matrix([[cos(theta),sin(theta)],
            [-sin(theta),cos(theta)]])
F = P*R
point(F,xmin=-10,xmax=10,ymin=-10,ymax=10,size=50,aspect_ratio=1)

    I think you might appreciate seeing points of matrix \(P\) alongside points of matrix \(F\). You can get that by running the code shown below:

    P = matrix([[-7,6],[3,1],[-3,-3],[6,-1],[2,-5],[-5,3],[-6,-7],[3,-3],[-1,-5],[-1,-2]])
theta = 3.07
R = matrix([[cos(theta),sin(theta)],
            [-sin(theta),cos(theta)]])
F = P*R
G = Graphics()
G += point(P,hue=0,xmin=-10,xmax=10,ymin=-10,ymax=10,size=50,aspect_ratio=1)
G += point(F,hue=0.7,size=50)
show(G)
