  1. Question

    Three people are buying fruit, and there are five grocery stores. Each person has a shopping list dictating how many pounds of each fruit to buy. Each store has different unit costs of fruit.

    In order to consider the total cost for each person at each store, we can use matrix multiplication. In order to help visualize matrix multiplication, the shopping lists are shown as a matrix on the bottom-left, the unit prices as a matrix on top-right, and the resulting (unfilled) matrix of totals in the bottom-right.

    A peculiarity of matrix multiplication requires the columns of the first matrix to “match” the rows of the second matrix. In this case, notice the various fruits correspond to columns in the first matrix (bottom-left) and rows in the second matrix (top-right). This visualization is an attempt to make this seem more natural.


    What is the unit cost of bananas, in dollars per pound, at Lovely Lunches?


    Solution

    You can use any person’s shopping trip at Lovely Lunches. Use \(x\) to represent the unknown unit cost.

    • If you use Quintin, solve the following equation:

    \[(4\cdot 1.87)+(2\cdot x)+(1\cdot 7.39)+(5\cdot 7.27)=52.6\]

    • If you use Rachel, solve the following equation:

    \[(2\cdot 1.87)+(1\cdot x)+(4\cdot 7.39)+(5\cdot 7.27)=70.34\]

    • If you use Sebastian, solve the following equation:

    \[(5\cdot 1.87)+(1\cdot x)+(1\cdot 7.39)+(2\cdot 7.27)=31.97\]

    Any equation solves to give \(x=0.69\). Thus, the unit price of bananas at Lovely Lunches is $0.69 per pound.
