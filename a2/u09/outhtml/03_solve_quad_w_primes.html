  1. Question

    By factoring the quadratic expression, find all values of \(x\) that make the equation true. In other words, solve the equation.

    \[{2}x^{2}-{7}x+{5}~=~0\]


    1. \(x=1\)
    2. \(x=-1\)
    3. \(x=\frac{1}{2}\)
    4. \(x=\frac{-1}{2}\)
    5. \(x=\frac{1}{3}\)
    6. \(x=\frac{-1}{3}\)
    7. \(x=\frac{1}{5}\)
    8. \(x=\frac{-1}{5}\)
    9. \(x=\frac{1}{7}\)
    10. \(x=\frac{-1}{7}\)
    11. \(x=2\)
    12. \(x=-2\)
    13. \(x=\frac{2}{3}\)
    14. \(x=\frac{-2}{3}\)
    15. \(x=\frac{2}{5}\)
    16. \(x=\frac{-2}{5}\)
    17. \(x=\frac{2}{7}\)
    18. \(x=\frac{-2}{7}\)
    19. \(x=3\)
    20. \(x=-3\)
    21. \(x=\frac{3}{2}\)
    22. \(x=\frac{-3}{2}\)
    23. \(x=\frac{3}{5}\)
    24. \(x=\frac{-3}{5}\)
    25. \(x=\frac{3}{7}\)
    26. \(x=\frac{-3}{7}\)
    27. \(x=5\)
    28. \(x=-5\)
    29. \(x=\frac{5}{2}\)
    30. \(x=\frac{-5}{2}\)
    31. \(x=\frac{5}{3}\)
    32. \(x=\frac{-5}{3}\)
    33. \(x=\frac{5}{7}\)
    34. \(x=\frac{-5}{7}\)
    35. \(x=7\)
    36. \(x=-7\)
    37. \(x=\frac{7}{2}\)
    38. \(x=\frac{-7}{2}\)
    39. \(x=\frac{7}{3}\)
    40. \(x=\frac{-7}{3}\)
    41. \(x=\frac{7}{5}\)
    42. \(x=\frac{-7}{5}\)

    Solution

    Factor the quadratic expression. Notice both the leading coefficient (\(2\)) and the constanct (\(5\)) are prime. This suggests we only need to guess and check a few possible factorizations. You should end up with the following equation:

    \[({2}x-{5})(x-{1})~=~0\]

    Which you could check by FOILing…

    To find the solutions, use the zero-product property.

    First solution

    \[({2}x-{5})=0\] \[x = \frac{5}{2}\]

    Second solution

    \[(x-{1})=0\] \[x = 1\]
