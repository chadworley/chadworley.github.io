  1. Question

    A point is on the unit circle and in quadrant II. A line connecting the point to the origin has a slope of -6.36. Find the coordinates, \((x,y)\), of the point.

    • \(x=\)
    • \(y=\)


    Solution

    Because the slope is -6.36, we know \(y=-6.36x\). Draw a sketch.


    The point is on the unit circle, so we know \(x^2+y^2=1\).

    \[x^2+(-6.36x)^2=1\]

    Distribute the power over the product.

    \[x^2+40.4496x^2=1\]

    Combine like terms.

    \[41.4496x^2=1\]

    Divide both sides by 41.4496.

    \[x^2=\frac{1}{41.4496}\]

    Square root both sides. We need to consider both the positive and negative possibilities.

    \[x=\pm \sqrt{\frac{1}{41.4496}}\]

    Since the point is in quadrant II, we know x is negative. We can get the decimal approximation of \(x\).

    \[x=-0.1553245\]

    Calculate \(y\) using \(y=(-6.36)\cdot(-0.1553245)\).

    \[y=0.9878635\]
