  1. Question

    In a forest diagram, there are 2 complete m-ary trees. Each tree has a branching factor of 3 and a maximum depth of 8.

    How many nodes are in the forest diagram? In other words, what is the size of the forest?


    Solution

    We can recognize that the number of nodes can be represented with a geometric series.

    \[S~=~2(3)^0+2(3)^1+2(3)^2+2(3)^3+2(3)^4+2(3)^5+2(3)^6+2(3)^7+2(3)^8\]

    The first term is \(a=2\). The common ratio is \(r=3\). The number of terms is \(n=9\). We can use the partial geometric series formula.

    \[S~=~\frac{a(1-r^n)}{1-r}\]

    \[S~=~\frac{2\left(1-3^{9}\right)}{1-3}\]

    \[S~=~19682\]
