  1. Question

    Consider the geometric series with the following recursive definition:

    \[f(n)~=~0.872\cdot f(n-1)\] \[f(1)~=~1\]

    Find the sum of the first 15 terms of the series.

    \[\sum_{n=1}^{15}f(n) ~=~ ?\]

    The tolerance is \(\pm 0.1\).


    Solution

    In Desmos, you can type in the first two equations to define the sequence recursively. Click Create Table and generate the first 15 terms:

    \[f(1)~=~1\] \[f(2)~=~0.872\] \[f(3)~=~0.760384\] \[f(4)~=~0.663054848\] \[f(5)~=~0.578183827456\] \[f(6)~=~0.504176297541632\] \[f(7)~=~0.439641731456303\] \[f(8)~=~0.383367589829896\] \[f(9)~=~0.33429653833167\] \[f(10)~=~0.291506581425216\] \[f(11)~=~0.254193739002788\] \[f(12)~=~0.221656940410431\] \[f(13)~=~0.193284852037896\] \[f(14)~=~0.168544390977045\] \[f(15)~=~0.146970708931984\]

    Now, you just need to add all 15 terms.

    \[f(1)+f(2)+f(3)+f(4)+f(5)+f(6)+f(7)+f(8)+f(9)+f(10)+f(11)+f(12)+f(13)+f(14)+f(15) ~=~ 6.811262\]

    Luckily, Desmos understands Capital-sigma notation! To get the upper-case sigma, which means “sum”, you can type sum. You should be able to type the expression using the summation operator.
