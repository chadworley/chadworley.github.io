15_geometric_value_from_index


  1. Question

    A geometric sequence has a common ratio between subsequent terms (remember an arithmetic progression has a common difference between terms).

    Let \(g_{n}\) represent the \(n\)th value in a geometric progression with the following recursive definition: \[g_1 = 9\] \[g_{[n+1]} = g_{[n]}\cdot 5\] where \(n\) is any natural number. Using this recursive definition can produce the next few values. \[g_2 ~=~ g_1 \cdot 5 ~=~ 9\cdot5 ~=~ 45\] \[g_3 ~=~ g_2 \cdot 5 ~=~ 45\cdot5 ~=~ 225\] \[g_4 ~=~ g_3 \cdot 5 ~=~ 225\cdot5 ~=~ 1125\]

    In general, if a geometric progression has an initial term \(g_1\) and a common ratio \(r\), we can write an explicit formula to find the \(n\)th value.

    \[g_n = g_1r^{n-1} \]

    • Evaluate \(g_{8}\)

    Solution

    Use the explicit formula.

    \[g_n = g_1r^{n-1} \]

    \[g_{8} = (9)(5)^{8-1} \]

    \[g_{8} = (9)(5)^{7} \]

    \[g_{8} = (9)(78125) \]

    \[g_{8} = 703125 \]
