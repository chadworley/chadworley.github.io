  1. Question

    A red triangle has coordinates \(A(-5.1, -5)\), \(B(-4.68, -7.83)\), and \(C(-3.87, -5.04)\). The triangle is rotated 3.55 radians counterclockwise around the origin to produce a blue triangle \(A'B'C'\).


    In general, if point \((x,y)\) is rotated by \(\theta\) radians (and clockwise is negative while counterclockwise is positive), the new point \((x',y')\) can be determined with the following equations:

    \[x' = x \cos(\theta) - y\sin(\theta)\] \[y' = x \sin(\theta) + y\cos(\theta)\]

    Find the \(x\) coordinate of point C. The tolerance is \(\pm0.01\).


    Solution

    \[x' ~=~ -3.87 \cos(3.55) - (-5.04)\sin(3.55) ~\approx~ 1.5500832\] \[y' ~=~ -3.87 \sin(3.55) + (-5.04)\cos(3.55) ~\approx~ 6.1624461\]

    So, the answer is 1.5500832.
