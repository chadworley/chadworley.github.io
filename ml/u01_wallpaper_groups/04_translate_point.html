Question

A p1 wallpaper was made by using repeated translations. I have depicted the translation of point \((0.41, -0.55)\) in the direction \(\theta=1.13\) (radians) by a distance of \(r=2.6\).


What are the coordinates of the point at the end of the arrow (the result of the translation)?

  • The horizontal coordinate, \(x'=\) ##ANSWER1##
  • The vertical coordinate, \(y'=\) ##ANSWER2##

Answerlist

  • 1.4
  • 1.43
  • 1.46
  • 1.49
  • 1.52
  • 1.74
  • 1.77
  • 1.8
  • 1.83
  • 1.86

Solution

To find the changes of translation, we can use trigonometry.

\[\Delta x = r\cdot \cos(\theta)\] \[\Delta y = r\cdot \sin(\theta)\]

Lets evaluate those changes.

\[\Delta x ~=~ (2.6)\cos(1.13) ~=~ 1.11\] \[\Delta y ~=~ (2.6)\sin(1.13) ~=~ 2.35\]

We can then add these changes to the original point.

\[x' ~=~ 0.41+1.11 ~=~ 1.52\] \[y' ~=~ -0.55+2.35 ~=~ 1.8\]
