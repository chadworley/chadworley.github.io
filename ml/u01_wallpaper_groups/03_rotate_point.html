Question

A red triangle has coordinates \(A(-1.4, -4.99)\), \(B(0.08, -7.41)\), and \(C(-1.78, -8.09)\). The triangle is rotated 2.19 radians clockwise around the origin to produce a blue triangle \(A'B'C'\).


In general, if point \((x,y)\) is rotated by \(\theta\) radians (and clockwise is negative while counterclockwise is positive), the new point \((x',y')\) can be determined with the following equations:

\[x' = x \cos(\theta) - y\sin(\theta)\] \[y' = x \sin(\theta) + y\cos(\theta)\]

Find the \(y\) coordinate of point C’. The tolerance is \(\pm0.01\).

Solution

\[x' ~=~ -1.78 \cos(-2.19) - (-8.09)\sin(-2.19) ~\approx~ -5.5549292\] \[y' ~=~ -1.78 \sin(-2.19) + (-8.09)\cos(-2.19) ~\approx~ 6.1448565\]

So, the answer is 6.1448565.
