  1. Question

    Let function \(f\) be defined as follows: \[f(x) = - 5 x^{2} - 2 x + 5\] Evaluate \(f(-5.97)\).

    You can round (or truncate) at the tenths place.


    Solution

    Replace all \(x\) variables with \(-5.97\) in the definition of \(f\).

    \[- 5 (-5.97)^{2} - 2 (-5.97) + 5\]

    You’ll want to use a calculator.

    \[f(-5.97)~=~-161.2645\]
