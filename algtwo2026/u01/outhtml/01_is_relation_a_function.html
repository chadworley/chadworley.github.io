01_is_relation_a_function


  1. Question

    A list of inputs and outputs is expressed as \((x,y)\) pairs.

    ( 5 , 8 )

    ( 8 , 7 )

    ( 2 , 8 )

    ( 2 , 8 )

    ( 1 , 4 )

    Is this list consistent with \(y\) being a function of \(x\)?

    Is this list consistent with \(x\) being a function of \(y\)?

    Is this list consistent with a one-to-one function?



    Solution

    Every repeated \(x\) value is matched with a consistent (also repeating) \(y\), so \(y\) is a function of \(x\).

    A \(y\) value repeates with different \(x\) values, so \(x\) is not a function of \(y\). For example: (5, 8) and (2, 8)

    Because \(x\) is not a function of \(y\), this list is inconsistent with a one-to-one function.



  2. Question

    A list of inputs and outputs is expressed as \((x,y)\) pairs.

    ( 9 , 6 )

    ( 8 , 8 )

    ( 1 , 8 )

    ( 2 , 3 )

    ( 7 , 3 )

    ( 9 , 2 )

    ( 4 , 9 )

    Is this list consistent with \(y\) being a function of \(x\)?

    Is this list consistent with \(x\) being a function of \(y\)?

    Is this list consistent with a one-to-one function?



    Solution

    An \(x\) value repeats with different \(y\) values, so \(y\) is not a function of \(x\). For example: (9, 6) and (9, 2)

    Also, a \(y\) value repeates with different \(x\) values, so \(x\) is not a function of \(y\). For example: (2, 3) and (7, 3)

    We would need both to be function of the other to have a one-to-one function. Neither is, so we donâ€™t even have a function in either direction.



  3. Question

    Is this list consistent with \(y\) being a function of \(x\)?
