  1. Question

    Consider the following product of linear binomials:

    \[({9}x-{4})({-7}x-{1})\]

    When the expression is expanded and like terms are combined, an equivalent expression can be expressed in standard form:

    \[ax^2+bx+c\]

    where \(a\), \(b\), and \(c\) are integers.

    • What is the value of \(c\)?

    Solution

    You could use the box method.


    Or, you can FOILâ€¦


    Either way, you should first get \[{-63}x^{2}-{9}x+{28}x+{4}\]

    And, after combining like terms, you should get \[{-63}x^{2}+{19}x+{4}\]

    Thus, \(a=-63\) and \(b=19\) and \(c=4\). This question specifically asked for the value of \(c\), so the answer is \(4\).
