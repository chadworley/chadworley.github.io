    Solution

    Undo the squaring on both sides. Notice that \(\sqrt{\frac{361}{4}}=\frac{\sqrt{361}}{\sqrt{4}}=\frac{19}{2}\). Also, remember that when you undo the squaring function, we need to consider multiple possibilities because the squaring function is NOT one-to-one.

    \[x+\frac{15}{2}~=~\pm\frac{19}{2}\]

    Subtract \(\frac{15}{2}\) from both sides of the equation.

    \[x~=~\frac{-15}{2}\pm\frac{19}{2}\]

    We are adding/subtracting fractions, and the fractions have a common denominator, so we can simply add/subtract the numerators.

    \[x~=~\frac{-15\pm 19}{2}\]

    Evaluate both solutions.

    \[x=-17~~~~~\text{or}~~~~x=2\]
