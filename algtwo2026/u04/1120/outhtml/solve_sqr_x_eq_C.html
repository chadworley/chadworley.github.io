    Solution

    Notice, because the squaring function is NOT one-to-one, when we “undo” it on both sides of an equation, we need to consider multiple possibilities.

    In this case, you should know that, when squared, both \(-4\) and \(4\) produce \(16\).

    \[(x+{7})^2~=~16\]

    \[x+{7}~=~\pm4\]

    \[x~=~-7\pm4\]

    So the two solutions are \(x=-3\) and \(x=-11\).
