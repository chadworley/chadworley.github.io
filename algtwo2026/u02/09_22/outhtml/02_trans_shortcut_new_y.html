  1. Question

    Consider the two functions \(f\) and \(g\), where \(g\) is defined as a transformation of \(f\), as shown below:

    \[g[x]~=~9\cdot f{\left[3\cdot x+6\right]}-4\]

    If we know values for \(x_1\) and \(y_1\) such that \(f[x_1]=y_1\), we can find values for \(x_2\) and \(y_2\) such that \(g[x_2]=y_2\).

    • What is the formula to calculate \(y_2\) from \(y_1\)?

    1. \(y_2~=~\frac{y_1}{9}+4\)
    2. \(y_2~=~4\cdot y_1-9\)
    3. \(y_2~=~9\cdot\left(y_1+4\right)\)
    4. \(y_2~=~\frac{y_1}{4}-9\)
    5. \(y_2~=~\frac{y_1+9}{4}\)
    6. \(y_2~=~4\cdot\left(y_1+9\right)\)
    7. \(y_2~=~4\cdot\left(y_1-9\right)\)
    8. \(y_2~=~4\cdot y_1+9\)
    9. \(y_2~=~\frac{y_1}{9}-4\)
    10. \(y_2~=~9\cdot\left(y_1-4\right)\)
    11. \(y_2~=~\frac{y_1-9}{4}\)
    12. \(y_2~=~\frac{y_1-4}{9}\)
    13. \(y_2~=~\frac{y_1}{4}+9\)
    14. \(y_2~=~9\cdot y_1-4\)
    15. \(y_2~=~\frac{y_1+4}{9}\)
    16. \(y_2~=~9\cdot y_1+4\)

    Solution

    If we force \(3\cdot x_2+6~=~x_1\), we will be able to set up the following equations:

    \[g[x_2]~=~9\cdot f{\left[3\cdot x_2+6\right]}-4\] \[g[x_2]~=~9\cdot f{\left[x_1\right]}-4\] \[y_2~=~9\cdot y_1-4\]

    Notice, you just use the operations outside the function.
