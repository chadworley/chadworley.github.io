  1. Question

    Consider the two functions \(f\) and \(g\), where \(g\) is defined as a transformation of \(f\), as shown below as an equation true for any value of \(x\):

    \[g[x]~=~2\cdot f{\left[8\left(x+3\right)\right]}-6\]

    If values \(x_{\text{old}}=88\) and \(y_{\text{old}}=43\) satisfy the equation \(y_{\text{old}}=f[x_{\text{old}}]\), we can find corresponding values for \(x_{\text{new}}\) and \(y_{\text{new}}\) such that \(y_{\text{new}}=g[x_{\text{new}}]\).

    • What is the value of \(x_{\text{new}}\)?

    Solution

    Use the shortcut, which tells us we can make a formula by inverting the argument expression. The argument expression says to add 3 and then multiply by 8, so the inverse must divide by 8 and then subtract 3.

    \[x_{\text{new}}~=~\frac{x_{\text{old}}}{8}-3\]

    \[x_{\text{new}}~=~\frac{88}{8}-3\]

    \[x_{\text{new}}~=~8\]
