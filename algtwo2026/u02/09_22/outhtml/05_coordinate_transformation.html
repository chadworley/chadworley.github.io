  1. Question

    Consider the two functions \(f\) and \(g\), where \(g\) is defined as a transformation of \(f\), as shown below as an equation true for any value of \(x\):

    \[g[x]~=~2\cdot f{\left[8\left(x+3\right)\right]}-6\]

    For every point \((a,b)\) on the curve \(y=f[x]\) there is a corresponding point on the curve \(y=g[x]\). What is the rule that describes this coordinate transformation?


    1. \((a,b)\to \left(\frac{a-3}{8},2 b-6\right)\)
    2. \((a,b)\to \left(\frac{a-8}{3},\frac{b}{2}+6\right)\)
    3. \((a,b)\to \left(\frac{a-8}{3},\frac{b}{6}+2\right)\)
    4. \((a,b)\to \left(\frac{a}{8}-3,2 b-6\right)\)
    5. \((a,b)\to \left(\frac{a}{8}-3,6 b-2\right)\)
    6. \((a,b)\to \left(3 a+8,\frac{b}{2}+6\right)\)
    7. \((a,b)\to \left(8 a+3,\frac{b+6}{2}\right)\)
    8. \((a,b)\to \left(8 a+3,6 b-2\right)\)
    9. \((a,b)\to \left(8\left(a+3\right),\frac{b}{6}+2\right)\)
    10. \((a,b)\to \left(8\left(a+3\right),2\left(b-6\right)\right)\)
    11. \((a,b)\to \left(8\left(a+3\right),6 b-2\right)\)
    12. \((a,b)\to \left(8\left(a+3\right),6\left(b-2\right)\right)\)

    Solution

    We know we need the argument \(8\left(x+3\right)\) to equal \(a\) because we wish to use the fact that \(f[a]=b\).

    \[8\left(x+3\right)~=~a\] Solve for \(x\).

    1. divide by 8
    2. subtract 3

    \[x~=~\frac{a}{8}-3\]

    So, now, since \(8\left(x+3\right)~=~a\), we can rewrite the main equation: \[g[x]~=~2\cdot f{\left[a\right]}-6\]

    We can also substitute \(\frac{a}{8}-3\) for \(x\) and substitute \(b\) for \(f[a]\).

    \[g{\left[\frac{a}{8}-3\right]}~=~2\cdot b-6\]

    And, thus, the corresponding point on the \(g\) curve is \(\left(\frac{a}{8}-3,2 b-6\right)\). As a coordinate transformation:

    \[(a,b)\to \left(\frac{a}{8}-3,2 b-6\right)\]

    \[g[x]~=~7\cdot\left(f{\left[\frac{x}{2}+4\right]}+5\right)\]

    \[g[x]~=~9\cdot f{\left[\frac{x}{6}-2\right]}-7\]

    For every point \((a,b)\) on the curve \(y=f[x]\) there is a corresponding point on the curve \(y=g[x]\). What is the rule that describes this coordinate transformation?


    1. \((a,b)\to \left(\frac{a-2}{6},\frac{b}{7}+9\right)\)
    2. \((a,b)\to \left(\frac{a-2}{6},\frac{b+7}{9}\right)\)
    3. \((a,b)\to \left(\frac{a-6}{2},\frac{b}{7}+9\right)\)
    4. \((a,b)\to \left(\frac{a-6}{2},\frac{b+9}{7}\right)\)
    5. \((a,b)\to \left(\frac{a}{2}-6,\frac{b}{7}+9\right)\)
    6. \((a,b)\to \left(\frac{a}{6}-2,\frac{b}{9}+7\right)\)
    7. \((a,b)\to \left(\frac{a}{6}-2,7 b-9\right)\)
    8. \((a,b)\to \left(\frac{a}{6}-2,9 b-7\right)\)
    9. \((a,b)\to \left(2 a+6,\frac{b}{7}+9\right)\)
    10. \((a,b)\to \left(2 a+6,\frac{b+9}{7}\right)\)
    11. \((a,b)\to \left(6 a+2,\frac{b}{7}+9\right)\)
    12. \((a,b)\to \left(6\left(a+2\right),9 b-7\right)\)

    Solution

    We know we need the argument \(\frac{x}{6}-2\) to equal \(a\) because we wish to use the fact that \(f[a]=b\).

    \[\frac{x}{6}-2~=~a\] Solve for \(x\).

    1. add 2
    2. multiply by 6

    \[x~=~6\left(a+2\right)\]

    So, now, since \(\frac{x}{6}-2~=~a\), we can rewrite the main equation: \[g[x]~=~9\cdot f{\left[a\right]}-7\]

    We can also substitute \(6\left(a+2\right)\) for \(x\) and substitute \(b\) for \(f[a]\).

    \[g{\left[6\left(a+2\right)\right]}~=~9\cdot b-7\]

    And, thus, the corresponding point on the \(g\) curve is \(\left(6\left(a+2\right),9 b-7\right)\). As a coordinate transformation:

    \[(a,b)\to \left(6\left(a+2\right),9 b-7\right)\]



  12. Question

    Consider the two functions \(f\) and \(g\), where \(g\) is defined as a transformation of \(f\), as shown below as an equation true for any value of \(x\):

    \[g[x]~=~2\cdot\left(f{\left[4 x+9\right]}+6\right)\]

    For every point \((a,b)\) on the curve \(y=f[x]\) there is a corresponding point on the curve \(y=g[x]\). What is the rule that describes this coordinate transformation?


    1. \((a,b)\to \left(\frac{a-9}{4},2\left(b+6\right)\right)\)
    2. \((a,b)\to \left(\frac{a}{4}-9,\frac{b}{2}-6\right)\)
    3. \((a,b)\to \left(\frac{a}{4}-9,2 b+6\right)\)
    4. \((a,b)\to \left(\frac{a}{9}-4,\frac{b}{2}-6\right)\)
    5. \((a,b)\to \left(\frac{a}{9}-4,2\left(b+6\right)\right)\)
    6. \((a,b)\to \left(\frac{a}{9}-4,6 b+2\right)\)
    7. \((a,b)\to \left(4 a+9,\frac{b-2}{6}\right)\)
    8. \((a,b)\to \left(4 a+9,\frac{b-6}{2}\right)\)
    9. \((a,b)\to \left(4 a+9,2\left(b+6\right)\right)\)
    10. \((a,b)\to \left(9\left(a+4\right),\frac{b-6}{2}\right)\)
    11. \((a,b)\to \left(9\left(a+4\right),\frac{b}{2}-6\right)\)
    12. \((a,b)\to \left(9\left(a+4\right),6\left(b+2\right)\right)\)

    Solution

    We know we need the argument \(4 x+9\) to equal \(a\) because we wish to use the fact that \(f[a]=b\).

    \[4 x+9~=~a\] Solve for \(x\).

    1. subtract 9
    2. divide by 4

    \[x~=~\frac{a-9}{4}\]

    So, now, since \(4 x+9~=~a\), we can rewrite the main equation: \[g[x]~=~2\cdot\left(f{\left[a\right]}+6\right)\]

    We can also substitute \(\frac{a-9}{4}\) for \(x\) and substitute \(b\) for \(f[a]\).

    \[g{\left[\frac{a-9}{4}\right]}~=~2\cdot\left(b+6\right)\]

    And, thus, the corresponding point on the \(g\) curve is \(\left(\frac{a-9}{4},2\left(b+6\right)\right)\). As a coordinate transformation:

    \[(a,b)\to \left(\frac{a-9}{4},2\left(b+6\right)\right)\]
