  1. Question

    If \(f[85]=67\), then there exists a knowable solution to the equation below.

    \[y=\frac{f{\left[5\left(x+6\right)\right]}-59}{2}\]

    Find the solution.

    • \(x=\)
    • \(y=\)


    Solution

    Rewrite the given information:

    \[f[85]=67\]

    \[y=\frac{f{\left[5\left(x+6\right)\right]}-59}{2}\]

    First, let’s make the argument equal to 85.

    \[5\left(x+6\right)~=~85\]

    Divide by 5.

    \[x+6~=~17\]

    Subtract 6.

    \[x~=~11\]

    Let’s plug that value of \(x\) into the given equation.

    \[y=\frac{f{\left[5\left((11)+6\right)\right]}-59}{2}\]

    Simplify the argument using arithmetic and order of operations.

    \[y=\frac{f{\left[85\right]}-59}{2}\]

    Excellent! Now we can use the first bit of information given to us: \(f[85]=67\). So, we can substitute 67 in place of \(f[85]\).

    \[y=\frac{67-59}{2}\]

    Simplify the expression using arithmetic and order of operations.

    \[y=4\]

    And, thus, we have our solution: \(x=11\) and \(y=4\).
