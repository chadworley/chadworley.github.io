  1. Question

    (Note, we use subscripts like adjectives. So, \(x_1\) is “first \(x\)” and \(x_2\) is “second \(x\)”. This means the subscript is not involved with any arithmetic; it just allows us to distiguish between different \(x\) values. The alternative would be to use a bunch of different letters, but then it is easy to forget which letters are \(x\) values.)

    Consider the two functions \(f\) and \(g\), where \(g\) is defined as a transformation of \(f\), as shown below:

    \[g[x]~=~9\cdot f{\left[3\cdot x+6\right]}-4\]

    If we know values for \(x_1\) and \(y_1\) such that \(f[x_1]=y_1\), we can find values for \(x_2\) and \(y_2\) such that \(g[x_2]=y_2\).

    • What is the formula to calculate \(x_2\) from \(x_1\)?

    1. \(x_2~=~\frac{x_1}{3}+6\)
    2. \(x_2~=~6\cdot x_1-3\)
    3. \(x_2~=~3\cdot\left(x_1+6\right)\)
    4. \(x_2~=~\frac{x_1}{6}-3\)
    5. \(x_2~=~\frac{x_1+3}{6}\)
    6. \(x_2~=~6\cdot\left(x_1+3\right)\)
    7. \(x_2~=~6\cdot\left(x_1-3\right)\)
    8. \(x_2~=~6\cdot x_1+3\)
    9. \(x_2~=~\frac{x_1}{3}-6\)
    10. \(x_2~=~3\cdot\left(x_1-6\right)\)
    11. \(x_2~=~\frac{x_1-3}{6}\)
    12. \(x_2~=~\frac{x_1-6}{3}\)
    13. \(x_2~=~\frac{x_1}{6}+3\)
    14. \(x_2~=~3\cdot x_1-6\)
    15. \(x_2~=~\frac{x_1+6}{3}\)
    16. \(x_2~=~3\cdot x_1+6\)

    Solution

    Let’s try to memorize the shortcut. To find the new \(x\) values, we use the INVERSE of the argument expression. In the \(f\) argument, the operations are multiply by 3 and then add 6. So, the inverse must subtract 6 and then divide by 3.

    \[x_2 ~=~ \frac{x_1-6}{3}\]

    The reason this works can be thought of this way… but it is quite abstract. We will practice with more specific examples soon.

    • We know that we want the equation’s left side to be \(g[x_2]\). This is because we are deriving an equation involving \(g[x_2]\). \[g[x_2]~=~9\cdot f{\left[3\cdot x_2+6\right]}-4\]

    • We only know the output of \(f\) if the input is \(x_1\). So, we need the \(f\) argument to equal \(x_1\). \[3\cdot x_2+6~=~x_1\]

    • But we want a formula solved for \(x_2\), so we’ll need to do some algebra. Subtract 6 from both sides. \[3x_2~=~x_1-6\]

    • Divide both sides by 3. \[x_2 ~=~ \frac{x_1-6}{3}\]

    As a final note, notice that if \(3\cdot x_2+6~=~x_1\), we will be able to set up the following equations:

    \[g[x_2]~=~9\cdot f{\left[x_1\right]}-4\]

    \[y_2~=~9\cdot y_1-4\]
