Question

In the graph below, we see \(y=f(x)\) as a blue line and \(y=g(x)\) as a red line.


We can see that \(g\) is a translated version of \(f\). Which equation describes this transformation?

Answerlist

  • \(g(x)~=~f(x-2)-7\)
  • \(g(x)~=~f(x-2)+7\)
  • \(g(x)~=~f(x-7)-2\)
  • \(g(x)~=~f(x-7)+2\)
  • \(g(x)~=~f(x+2)-7\)
  • \(g(x)~=~f(x+2)+7\)
  • \(g(x)~=~f(x+7)-2\)
  • \(g(x)~=~f(x+7)+2\)

Solution

To go from blue dots to red dots, you go 2 units right and 7 units up.


So, \[g(x)~=~f(x-2)+7\]
