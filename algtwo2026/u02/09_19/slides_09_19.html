An example:

Curve \(y=f[x]\) contains points \((3,1)\) and \((5,7)\) and \((7,4)\), as shown on the \(x\)-\(y\) plane below.

In other words: \[f[3]=1\] \[f[5]=7\] \[f[7]=4\]

Let function \(g\) be a transformation of function \(f\), as defined by the equation below:

\[g[x]~=~2\cdot f{\left[\frac{x}{3}+3\right]}-2\]

Find the three corresponding points of curve \(y=g[x]\).

Solution

\(x\) \(g[x]\)
0 0
6 12
12 6

How to get the answers…

Rewrite the given information: \[f[3]=1\] \[f[5]=7\] \[f[7]=4\] \[g[x]=2\cdot f{\left[\frac{x}{3}+3\right]}-2\]

  1. Use \(f[3]=1\).
    • Set the argument of \(f\) equal to 3. \[\frac{x}{3}+3=3\] Solve the two-step equation. \[x=0\] Substitute 0 for \(x\) in the main equation. \[g[0]=2\cdot f{\left[\frac{0}{3}+3\right]}-2\] Simplify the argument. \[g[0]=3\cdot f{\left[3\right]}-3\] Substitute 1 for \(f[3]\) because \(f[3]=1\). \[g[0]=3\cdot 1-3\] Use arithmetic to evaluate the numerical expression. \[g[0]=0\] So our first new point on \(y=g(x)\) is point \((0,0)\).
  2. Use old point = \((5,7)\). So, use \(f[5]=7\).
    • Set the argument of \(f\) equal to 5. \[\frac{x}{3}+3=5\] Solve the two-step equation. \[x=6\] Substitute 6 for \(x\) in the main equation. \[g[6]=2\cdot f{\left[\frac{6}{3}+3\right]}-2\] Simplify the argument. \[g[6]=3\cdot f{\left[5\right]}-3\] Substitute 7 for \(f[5]\) because \(f[5]=7\). \[g[6]=3\cdot 7-3\] Use arithmetic to evaluate the numerical expression. \[g[6]=12\] So our second new point on \(y=g(x)\) is point \((6,12)\).
  3. Use old point = \((7,4)\). So, use \(f[7]=4\).
    • Set the argument of \(f\) equal to 7. \[\frac{x}{3}+3=7\] Solve the two-step equation. \[x=12\] Substitute 12 for \(x\) in the main equation. \[g[12]=2\cdot f{\left[\frac{12}{3}+3\right]}-2\] Simplify the argument. \[g[12]=3\cdot f{\left[7\right]}-3\] Substitute 4 for \(f[7]\) because \(f[7]=4\). \[g[12]=3\cdot 4-3\] Use arithmetic to evaluate the numerical expression. \[g[12]=6\] So our third new point on \(y=g(x)\) is point \((12,6)\).

Shortcut…

You can save a lot of time by recognizing the following:

  • The new \(x\) values are found using the INVERSE of the argument expression. Remember, the expression in the argument was \(\frac{x}{3}+3\), so:
    • \[x_{\text{new}}~=~3\cdot\left(x_{\text{old}}-3\right)\]
  • The new \(y\) values are found using the operations outside the function. It might help to see the original expression with an abbreviated argument: \(2\cdot f[\text{arg}]-2\)
    • \[y_{\text{new}}~=~2\cdot y_{\text{old}}-2\]
