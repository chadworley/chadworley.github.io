How to get the answers…

Rewrite the given information: \[f[0]=18\] \[f[12]=4\] \[f[18]=2\] \[g[x]=\frac{f{\left[3\cdot x-6\right]}}{2}+2\]

  1. Use \(f[0]=18\).
    • Set the argument of \(f\) equal to 0. \[3\cdot x-6=0\] Solve the two-step equation. \[x=2\] Substitute 2 for \(x\) in the main equation. \[g[2]=\frac{f{\left[3\cdot 2-6\right]}}{2}+2\] Simplify the argument. \[g[2]=\frac{f{\left[0\right]}}{2}+2\] Substitute 18 for \(f[0]\) because \(f[0]=18\). \[g[2]=\frac{18}{2}+2\] Use arithmetic to evaluate the numerical expression. \[g[2]=11\] So our first new point on \(y=g(x)\) is point \((2,11)\).
  2. Use old point = \((12,4)\). So, use \(f[12]=4\).
    • Set the argument of \(f\) equal to 12. \[3\cdot x-6=12\] Solve the two-step equation. \[x=6\] Substitute 6 for \(x\) in the main equation. \[g[6]=\frac{f{\left[3\cdot 6-6\right]}}{2}+2\] Simplify the argument. \[g[6]=\frac{f{\left[12\right]}}{2}+2\] Substitute 4 for \(f[12]\) because \(f[12]=4\). \[g[6]=\frac{4}{2}+2\] Use arithmetic to evaluate the numerical expression. \[g[6]=4\] So our second new point on \(y=g(x)\) is point \((6,4)\).
  3. Use old point = \((18,2)\). So, use \(f[18]=2\).
    • Set the argument of \(f\) equal to 18. \[3\cdot x-6=18\] Solve the two-step equation. \[x=8\] Substitute 8 for \(x\) in the main equation. \[g[8]=\frac{f{\left[3\cdot 8-6\right]}}{2}+2\] Simplify the argument. \[g[8]=\frac{f{\left[18\right]}}{2}+2\] Substitute 2 for \(f[18]\) because \(f[18]=2\). \[g[8]=\frac{2}{2}+2\] Use arithmetic to evaluate the numerical expression. \[g[8]=3\] So our third new point on \(y=g(x)\) is point \((8,3)\).


Shortcut…

You can save a lot of time by recognizing the following:

  • The new \(x\) values are found using the INVERSE of the argument expression. Remember, the expression in the argument was \(3\cdot x-6\), so:
    • \[x_{\text{new}}~=~\frac{x_{\text{old}}+6}{3}\]
  • The new \(y\) values are found using the operations outside the function. It might help to see the original expression with an abbreviated argument: \(\frac{f[\text{arg}]}{2}+2\)
    • \[y_{\text{new}}~=~\frac{y_{\text{old}}}{2}+2\]
