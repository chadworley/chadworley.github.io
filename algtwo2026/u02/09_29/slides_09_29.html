Shifts

  • Adding/subtracting in the argument shifts function horizontally. The direction is backwards.
  • Adding/subtracting outside the function shifts function vertically. The direction is normal.
  • Examples:
    • If we compare \(y=f(x)\) and \(y=f(x+2)+3\), the second curve is congruent to the first, but the second curve is shifted 2 units left and 3 units up.
    • If we compare \(y=f(x)\) and \(y=f(x-2)+3\), the second curve is congruent to the first, but the second curve is shifted 2 units right and 3 units up.
    • If we compare \(y=f(x)\) and \(y=f(x+2)-3\), the second curve is congruent to the first, but the second curve is shifted 2 units left and 3 units down.
    • If we compare \(y=f(x)\) and \(y=f(x-2)-3\), the second curve is congruent to the first, but the second curve is shifted 2 units right and 3 units down.

\[y=x^2\]

\[y=x^3\]

Exponential (base 2) function

\[y=2^x\]
