  1. Question

    Below are some common parent functions.

    names expressions
    absolute value \(f(x)=\lvert x \rvert\)
    quadratic \(f(x)=x^2\)
    cubic \(f(x)=x^3\)
    reciprocal \(f(x)=\frac{1}{x}\)
    square root \(f(x)=\sqrt{x}\)
    cube root \(f(x)=\sqrt[3]{x}\)
    exponential \(f(x)=2^{x}\)
    logarithmic \(f(x)=\log_{2}(x)\)

    One of those parent functions has been translated (shifted) to produce the graph below:


    • Which parent function was used?

    • What translation was used?



    Solution

    The equation used to produce the plot was: \[y~=~\lvert x-4 \rvert-1\]

    In other words, the parent function was the absolute value function: \(f(x)=\lvert x \rvert\). The parent function was shifted 4 units right and 1 units down, so the function-transformation equation was \(y = f(x-4)-1\).

    Here is a graph of the parent (in grey) and child (in blue).
