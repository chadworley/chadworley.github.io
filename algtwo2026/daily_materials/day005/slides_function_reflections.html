Puzzle 1

Let function \(f\) be defined by the table below.

\(x\) \(f(x)\)
-3 -16
-2 12
-1 15
0 11
1 -13
2 14
3 -10

Let \(a=-1\). Evaluate \(f(-a)\).

  • Solution
  • Since \(a=-1\), we know \(-a=1\). So we evaluate \(f(1)\). \[f(1) = -13\] Thus, \(f(-(-1))=-13\) and \[f(-a)=-13\]

Puzzle 2

Let function \(f\) be defined by the table below.

\(x\) \(f(x)\)

-1 -13
0 15

3 -11

Let \(a=-2\). Evaluate \(-f(a)\).

  • Solution
  • First, evaluate \(f(-2)\). \[f(-2)=10\]
  • So, we can just multiply both sides of the equation by \(-1\). \[-f(-2) = -10\]
  • Thus, \[-f(a)=-10\] (because \(a=-2\))

Summary

  • Graphs of \(y=f(x)\) and \(y=f(-x)\) are reflections over the \(y\)-axis. (So the corresponding \(x\) values are opposites.)
    • For example if point \((3,1)\) is on \(y=f(x)\) then point \((-3,1)\) is on \(y=f(-x)\).
  • Graphs of \(y=f(x)\) and \(y=-f(x)\) are reflections over the \(x\)-axis. (So the corresponding \(y\) values are opposites.)
    • For example if point \((3,1)\) is on \(y=f(x)\) then point \((3,-1)\) is on \(y=-f(x)\).
  • Graphs of \(y=f(x)\) and \(y=-f(-x)\) are reflections over both axes. In other words, they are 180Â° rotations around the origin. (So the corresponding \(x\) values are opposites, and the corresponding \(y\) values are opposites.)
    • For example if point \((3,1)\) is on \(y=f(x)\) then point \((-3,-1)\) is on \(y=-f(-x)\).
