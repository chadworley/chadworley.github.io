Question

Let function \(f\) be defined by the one-to-one piece-wise linear graph shown below.


Four reflections of \(f\) are graphed below (and \(y=f(x)\) is shown as a faint dashed line).


Match the graphs with their functions.

  • \(y=f(-x)\) is graph ##ANSWER1##
  • \(y=f^{-1}(x)\) is graph ##ANSWER2##
  • \(y=-f(-x)\) is graph ##ANSWER3##
  • \(y=-f(x)\) is graph ##ANSWER4##

Solution

Each of the graphs is a rigid transformation of the \(y=f(x)\) graph.

  • \(y=f(-x)\) is formed by a reflection over y axis, so all x values are multiplied by -1. So, graph C.
  • \(y=f^{-1}(x)\) is formed by a reflection over the y=x line, so x and y values are swapped. So, graph B.
  • \(y=-f(-x)\) is formed by reflections over x and y axes, so all x and y values are multiplied by -1. So, graph A.
  • \(y=-f(x)\) is formed by a reflection over x axis, so all y values are multiplied by -1. So, graph D.
