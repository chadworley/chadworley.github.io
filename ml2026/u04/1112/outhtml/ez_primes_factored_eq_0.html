  1. Question

    Find the values of \(x\) that make the following equation true. (In other words, solve the equation.)

    \[(2x-5)(7x-3)=0\]


    1. \(x=\frac{3}{2}\)
    2. \(x=\frac{-3}{2}\)
    3. \(x=\frac{5}{2}\)
    4. \(x=\frac{-5}{2}\)
    5. \(x=\frac{7}{2}\)
    6. \(x=\frac{-7}{2}\)
    7. \(x=\frac{2}{3}\)
    8. \(x=\frac{-2}{3}\)
    9. \(x=\frac{5}{3}\)
    10. \(x=\frac{-5}{3}\)
    11. \(x=\frac{7}{3}\)
    12. \(x=\frac{-7}{3}\)
    13. \(x=\frac{2}{5}\)
    14. \(x=\frac{-2}{5}\)
    15. \(x=\frac{3}{5}\)
    16. \(x=\frac{-3}{5}\)
    17. \(x=\frac{7}{5}\)
    18. \(x=\frac{-7}{5}\)
    19. \(x=\frac{2}{7}\)
    20. \(x=\frac{-2}{7}\)
    21. \(x=\frac{3}{7}\)
    22. \(x=\frac{-3}{7}\)
    23. \(x=\frac{5}{7}\)
    24. \(x=\frac{-5}{7}\)

    Solution

    Use the zero-product property. This tells us the solutions can be found by solving both of the following equations: \(2x-5=0\) and \(7x-3=0\).

    1. Start with the first one. \[2x-5=0\] Add 5 \[2x=5\] Divide both sides by 2. \[x=\frac{5}{2}\]

    2. Now do the second one. \[7x-3=0\] Add 3 \[7x=3\] Divide both sides by 7. \[x=\frac{3}{7}\]
