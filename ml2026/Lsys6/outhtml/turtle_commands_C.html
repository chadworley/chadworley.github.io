    Please use this L-system string generator along with Turbo Turtle to generate the pattern.

    With Turbo Turtle, you’ll want to “See Inside” to adjust the initial \(x\), initial \(y\), and DISTANCE variable to get the pattern centered and scaled. You’ll also want to turn on Turbo Mode (in the Edit drop-down menu).

    Solution

    The correct pattern is pattern A.

    1. Use the first link: https://chadworley.github.io/Lsys_string_gen.html
    2. Copy in the starting string, replacement rules, and number of recursions.
    3. Click Generate Strings
    4. Click Copy Last String
    5. Go to the Turbo Turtle: https://scratch.mit.edu/projects/1225522722/editor/
    6. In the code, set the ANGLE to goodies[[gc]][[4]].
    7. I like to start with DISTANCE equal to 1. I will make it larger or smaller as needed to change the scale of the pattern.
    8. Make sure to turn on Turbo Mode (in the Edit drop-down menu).
    9. Click the green flag.
    10. Paste the string.
    11. If necessary to keep the pattern centered and scaled, adjust the starting position, direction, and DISTANCE variable.
    12. Back to the quiz, select the matching pattern.
