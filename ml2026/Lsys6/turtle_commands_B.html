Question

Consider the \(L\)-system (and angle) defined below:

  • Start: A++A++A++A
  • Rules:
    A > B-CAC-B
    B > AB-BA
    C > CC
  • Number of recursions (number of iterations):
    • 5
  • Angle
    • 135

When we use the resulting string (and angle) to drive turtle graphics, using https://draemm.li/various/fractals/l-systems/, what image is produced?

You can also use this L-system string generator along with Turbo Turtle to generate the pattern.

(You will need to adjust other parameters to get image rotated, centered, and scaled.)

Solution

The correct pattern is pattern A.

  1. Use the link: https://draemm.li/various/fractals/l-systems/
  2. Copy in the starting string, replacement rules, number of recursions, and angle.
  3. Adjust the center and scale to clearly see the pattern. You might also need to rotate the pattern.
  4. Select the correct pattern.
